# Task-5-Exploratory-Data-Analysis-EDA-

##  Project Overview

This project was completed as part of the **Elevate Labs Data Analyst Internship – Task 5**.

The objective of this project is to perform **Exploratory Data Analysis (EDA)** on the Titanic dataset using Python. The analysis helps identify data patterns, missing values, feature relationships, and key insights through statistical summaries and visualizations.

---

##  Objectives

- Understand the dataset structure.
- Perform statistical analysis.
- Identify missing values.
- Analyze relationships between features.
- Visualize data using charts.
- Generate meaningful business insights.

---

##  Dataset Information

- **Dataset:** Titanic Dataset
- **Source:** Kaggle
- **Rows:** 891
- **Columns:** 12
- **File Type:** CSV

---

##  Exploratory Data Analysis Performed

### Data Exploration
- Loaded dataset using Pandas
- Displayed first five records
- Checked dataset information
- Generated statistical summary
- Identified missing values
- Counted unique categorical values

### Visualizations
- Histogram (Age Distribution)
- Countplot (Survival Count)
- Survival by Gender
- Survival by Passenger Class
- Boxplot (Age vs Passenger Class)
- Scatterplot (Age vs Fare)
- Correlation Heatmap

---

##  Key Findings

- The dataset contains **891 rows** and **12 columns**.
- Missing values were found in **Age, Cabin, and Embarked** columns.
- Female passengers had a higher survival rate than male passengers.
- First-class passengers had better survival chances.
- Passenger fare and class influenced survival probability.
- Most passengers were between **20–40 years** of age.
